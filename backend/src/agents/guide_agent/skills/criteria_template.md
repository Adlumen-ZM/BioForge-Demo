# HAp/矿化/肽段领域文献准入/排除标准模板
# 所属：backend/src/agents/guide_agent/skills/criteria_template.md
# 用途：提供 HAp/磷酸钙/牙体矿化领域的推荐文献准入排除标准，供 guide_agent 生成标准时参考

---

## 推荐准入标准（Inclusion Criteria）

以下标准适用于 HAp/矿化/肽段相互作用领域的系统综述和数据挖掘：

1. **研究对象为合成肽或天然蛋白来源肽段**（包括但不限于：矿化蛋白片段、人工合成短肽、功能性肽模拟物）
2. **实验涉及 HAp、磷酸钙（TCP/OCP/ACP）或牙体相关底物**（羟基磷灰石、牙釉质、牙本质、骨矿物质等）
3. **包含定性或定量的实验结果**（如结合亲和力、矿化促进/抑制率、SEM/XRD 表征数据等）
4. **英文原文**（有摘要和全文可获取）
5. **发表年份不早于 2000 年**（分子生物学和材料科学相关研究的有效年限参考）
6. **提供具体的肽段序列或化学结构信息**（可以是单字母缩写序列，如 WKYMVM，或 IUPAC 命名）
7. **报告了明确的实验模型**（如 SBF 浸泡实验、吸附等温线、AFM 力谱等）

---

## 推荐排除标准（Exclusion Criteria）

以下类型的文献应从分析中排除：

1. **综述、评论、会议摘要、编辑信函**（无原始实验数据）
2. **无实验数据的纯计算或纯分子动力学模拟研究**（除非研究目标明确为计算预测领域）
3. **研究对象明确排除肽段**（如仅研究无机矿物颗粒、有机小分子，与肽段无关）
4. **无法获取全文或摘要信息不完整**（无法判断是否满足准入标准）
5. **非人类/动物来源且无明确功能表征的天然蛋白**（过于宽泛，无法提取结构化数据）
6. **重复发表或预印本与正式发表版本重复**（保留引用次数更多的版本）

---

## 领域说明

> Demo 模板适用于 HAp/磷酸钙/牙体矿化/肽段相互作用领域的文献筛选。
>
> **screen_agent 将以此标准为基础**，对 search_agent 检索到的候选文献进行相关性筛选，
> 只保留满足准入标准、不触碰排除标准的文献。
>
> 本标准为 Demo 版默认模板；实际使用时，guide_agent 将根据用户的具体研究诉求，
> 在此模板基础上动态调整准入/排除条件。
